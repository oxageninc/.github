# .github

Organization-level defaults for [**Oxagen**](https://github.com/oxageninc).

- [`profile/README.md`](./profile/README.md) renders the public org page at
  [github.com/oxageninc](https://github.com/oxageninc).
- [`FUNDING.yml`](./FUNDING.yml) sets the org-wide **Sponsor** button
  ([sponsor Mac Anderson](https://github.com/sponsors/macanderson)).

Oxagen is the infrastructure layer for AI agents. Site: [oxagen.sh](https://oxagen.sh).
